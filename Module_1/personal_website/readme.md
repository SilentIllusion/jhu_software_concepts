# Personal Developer Website

A simple personal website template for developers.

## Files Included

1. `base.html` - Contains base code for every page
2. `home.html` - Home page 
3. `projects.html` - Projects page
4. `contact.html` - Contact page
5. `style.css` - CSS styles 
6. `main.py` - Uses Flask to set up the web app

## Features

- Clean, semantic HTML structure
- Bio section on the left
- Profile image on the right
- Responsive-ready structure

## Usage

1. Replace placeholder content with your information
2. Add your profile image as `profile-image.jpg`
3. Update `style.css` file for styling

## Customization

- Change name in the `<h1>` tag
- Update position in the `<h2>` tag
- Modify bio text in the `.bio-text` paragraphs
- Replace image source and alt text

## Requirements

- Modern web browser
- Python3
- Flask
- Basic text editor
